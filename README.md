# 如果当初买黄金

一个帮助记录冲动消费、换算成黄金价值的轻量网页应用。通过实时/手动录入今日金价，把曾经的消费金额折算为黄金克重，直观地看到如果当初买黄金，现在会值多少钱。

## 功能特性

- **今日金价**：尝试从 Yahoo Finance 获取 GC=F 期货的实时价格，并根据所选货币换算为每克金价；若接口不可用，可手动输入。
- **多次消费记录**：填写消费内容、金额、当时金价（可选）及日期，自动计算折算克重与今日价值。
- **累计统计**：展示所有记录的累计花费、折算克重、今日价值以及差额。
- **本地保存**：所有数据保存在浏览器 LocalStorage 中，可随时清空。
- **多货币切换**：支持 CNY、USD、HKD、EUR；切换时会清空现有记录，确保金额口径一致。

## 使用方式

1. 本项目为静态网页，直接打开 `index.html` 即可使用。
2. 若需要本地启动一个静态服务器，可使用任意工具，例如 Python 内置 HTTP 服务：

   ```bash
   python -m http.server 5173
   ```

   然后在浏览器访问 `http://localhost:5173`。

## 注意事项

- 外部接口若因网络或频率限制不可用，请手动输入当前金价。
- 汇率转换依赖 `open.er-api.com` 的公开接口，若获取失败，金额计算不会受影响，但需要手动设置金价。
- 本应用仅用于自省和估算，不构成投资建议。
